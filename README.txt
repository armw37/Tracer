# Tracer

A 2D particle tracing simulation in the Lagrangian sense. 
Given a set of x,y coordiantes and u,v component velocity vectors


## Usage

See Tracer/tracer_test.py for an example of how to run a simulatiom
See Tracer/plasma_sep2.txt for an example of the input coordinate
geometry and velocity.
